About Us

STim is an online game store that provides third party games from different publishers. It provides user with updated news of game updates, on-going event, and game exhibition launch. To support the publisher, all games must be purchased.
It was launched as a standalone game store in June 2021. We made this for project and provides a good online games store around the world. We run this with our community and make a creatures of games.
